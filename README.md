# Identity Sheet

A Python project for analyzing the impact of moving cells in a spreadsheet.

## Usage

1. Create an instance of the `IdentitySheet` class.
2. Add cells to the sheet using the `add_cell` method.
3. Move a cell using the `move_cell` method.
4. Get the dependent calculations for a cell using the `get_dependent_calculations` method.
5. Highlight the affected calculations using the `highlight_affected_calculations` method.
6. Summarize the impact of moving a cell using the `summarize_impact` method.
